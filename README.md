# تطبيق رسائل بسيط

تطبيق سطر أوامر صغير لإرسال رسائل نصية بين مستخدمين محلياً باستخدام ملف JSON.

## المتطلبات
- Python 3.10 أو أحدث (يعمل بدون أي تبعيات خارجية).

## طريقة الاستخدام
1. تسجيل المستخدمين:
   ```bash
   python main.py register اسم_المستخدم
   ```
2. إرسال رسالة:
   ```bash
   python main.py send اسم_المرسل اسم_المستلم "نص الرسالة"
   ```
3. عرض الرسائل الواردة:
   ```bash
   python main.py inbox اسم_المستخدم
   ```
4. عرض الرسائل المرسلة:
   ```bash
   python main.py sent اسم_المستخدم
   ```

يمكن تغيير مكان تخزين البيانات عبر تمرير المسار:
```bash
python main.py --store /tmp/messages.json register احمد
```

## الاختبارات
نفّذ الاختبارات بـ:
```bash
python -m unittest
```
