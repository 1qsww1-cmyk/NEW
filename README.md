# Edge Lite Optimizer

> "Transform Microsoft Edge from a resource-hungry browser into a lightning-fast, privacy-focused experience."

## Overview
Edge Lite is a powerful PowerShell script designed to optimize Microsoft Edge by stripping away unnecessary telemetry, background services, and bloatware. It forces the browser to run in a resource-efficient mode, significantly reducing RAM and CPU usage while hardening your privacy.

## Key Features
- **Performance Boost:** Disables Startup Boost, background pre-rendering, and unnecessary background tasks to free up RAM.
- **Privacy Hardening:** Implements strict tracking prevention, blocks WebRTC IP leaks, and enforces DNS over HTTPS.
- **Bloatware Removal:** Disables built-in "distractions" like Shopping Assistant, Sidebar, Wallet, and Rewards.
- **Efficiency:** Enables the browser's built-in "Efficiency Mode" for optimal battery and processor usage.

## How to use
1. Download the `OptimizeEdge.ps1` script.
2. Right-click the file and select **"Run with PowerShell"**.
   *(Note: You must run this with Administrator privileges).*
3. Wait for the process to complete.
4. Restart Microsoft Edge to apply all changes.

## Security Warning
This script modifies Windows Registry keys related to Microsoft Edge policies. While safe, please ensure you have reviewed the script content before execution.
